JETLUX — IMAGE FILES
====================
Place all aircraft and site photos in this /images/ folder.

Required files (match filenames exactly):
  fleet-cj4.jpg          Cessna Citation CJ4 exterior
  fleet-phenom.webp      Embraer Phenom 300E exterior
  fleet-hawker.webp      Hawker 900XP exterior
  fleet-gulfstream.jpg   Gulfstream G450 exterior
  fleet-bbj.jpg          Boeing Business Jet exterior
  hero-bg.jpg            Homepage hero background (full bleed)
  og-default.jpg         Open Graph image — 1200x630px

After uploading images here, update the src attributes in the HTML
from the external URLs to /images/filename.jpg
See data-local-path attribute on each <img> tag for the target filename.
